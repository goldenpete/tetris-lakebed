# Tetris Arena

A real-time multiplayer Tetris game with a competitive twist. Play as a guest or sign in to track your wins!

## How to Play

1. Click **Find Match** to enter the matchmaking queue
2. When an opponent is found, the 3-minute match begins automatically
3. Use arrow keys to move pieces, Up to rotate, Space to hard drop
4. Clear lines and score points - highest score wins when time runs out
5. Stack too high and you'll be disqualified!

## Features

- **Guest & Account Play**: Anyone can play, but only logged-in users track wins
- **Real-time Multiplayer**: See your opponent's board live as you play
- **3-Minute Matches**: Fast, intense games
- **Win Tracking**: Trophy badge shown for signed-in users
- **Cross-Platform**: Works on desktop with keyboard controls

## Run Locally

```sh
npx lakebed dev
```

Then open http://localhost:3000 and click **Find Match** to start playing!
